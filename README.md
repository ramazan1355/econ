ECO.N: Bio-Node Firmware
Прошивка для автономных узлов очистки воздуха ECO.N. Система контролирует состояние био-фильтра (мох) и управляет микроклиматом в городской среде.

Стек
Контроллер: ESP32

Периферия: DHT22 (климат), MQ-135 (качество воздуха), емкостный датчик влажности (состояние субстрата).

Исполнительные элементы: ШИМ-вентилятор, реле насоса.

Принцип работы
Код циклически опрашивает датчики и меняет режим работы в зависимости от условий:

ECO: Экономичный режим. 30% мощности вентилятора.

NORMAL: 60% мощности, активный полив.

MAX: 100% мощности, интенсивное охлаждение.

BOOST: Принудительная фильтрация при обнаружении загрязнений (AQI).

Запуск
Собрать схему согласно пинам в коде.

Прошить через Arduino IDE/PlatformIO.

Подключить питание 5V/12V (через понижающий модуль для ESP32).

Данные будут приходить в MQTT-топик econ/data.

Продажи и масштабирование
Целевая модель: B2G (контракты с городами) и B2B (девелоперы, установка в ЖК).

Запуск: Установка пилотного узла в парковой зоне или на территории бизнес-центра для сбора метрик (снижение PM2.5, снижение температуры).

Монетизация: Продажа оборудования + сервисное обслуживание био-картриджей.
